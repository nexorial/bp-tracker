# Screenshots

This directory contains screenshots of the BP Tracker application for documentation purposes.

## Required Screenshots

To complete the documentation, capture the following screenshots:

1. **dashboard.png** - Main dashboard showing charts, input form, and records list
2. **dashboard-overview.png** - Wide view of the entire dashboard
3. **add-reading.png** - Input form with validation visible
4. **records-list.png** - Records table with color-coded readings
5. **statistics.png** - Statistics panel with averages and trends
6. **export.png** - Export button and/or exported CSV file

## How to Capture

1. Run the application: `npm run dev`
2. Add several sample blood pressure readings
3. Use browser dev tools screenshot feature or system screenshot tool
4. Save images to this directory with the names listed above
5. Update the main README.md to reference the correct paths
